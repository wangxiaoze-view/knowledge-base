想要搭建一个`electron`项目可以根据[官网](https://www.electronjs.org/)的步骤一步步实现；如果出现问题可自行百度；

不过在这里我推荐使用一个脚手架工具[electron-vite](https://cn.electron-vite.org/guide/),它这个工具可以直接生成一套成熟稳定的`electron+vite`模板,但是想要使用

::: tip
注意：electron-vite 需要 Node.js 版本 18+，20+ 和 Vite 版本 4.0+
:::

## 如何使用

<code-group>
  <code-block title="NPM" active>
  ```bash
  npm create @quick-start/electron@latest
  ```
  </code-block>

  <code-block title="Yarn">
  ```bash
  yarn create @quick-start/electron
  ```
  </code-block>

  <code-block title="PNPM">
  ```bash
  pnpm create @quick-start/electron
  ```
  </code-block>

</code-group>

然后按照提示操作即可!

```bash
✔ Project name: … <electron-app>
✔ Select a framework: › vue
✔ Add TypeScript? … No / Yes
✔ Add Electron updater plugin? … No / Yes
✔ Enable Electron download mirror proxy? … No / Yes

Scaffolding project in ./<electron-app>...
Done.
```

你还可以通过附加的命令行选项直接指定项目名称和你想要使用的模板。例如，要构建一个 Electron + Vue 项目，运行:

``` sh
# npm 6.x
npm create @quick-start/electron my-app --template vue

# npm 7+, extra double-dash is needed:
npm create @quick-start/electron my-app -- --template vue

# yarn
yarn create @quick-start/electron my-app --template vue

# pnpm
pnpm create @quick-start/electron my-app --template vue
```

安装成功之后，可以看下目录接口


## 入口文件

``` json
{
  "name": "electron-app", // 应用名
  "version": "1.0.0", // 应用版本
  "main": "./out/main/index.js" // 入口文件
}
```

## 目录结构

```js
├──src/
│  ├──main // 主进程
│  ├──preload // 预加载
│  └──renderer // 渲染进程，也就是后期你vue的页面操作
├──electron.vite.config.js // electron的配置文件，可以理解在vue中就是vite.config.js
└──package.json
```

项目的基本搭建就完成了，后续文章会总结一些相关知识
