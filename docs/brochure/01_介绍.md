# 介绍

::: tip
`前端小册`记录个人在前端方向上的一些总结，从基础的`前端三件套`到项目再到服务端数据库，最后项目部署等等一系列整套前端开发流程；

这是一条快速入门前端路线，不仅涉及到前端的基础知识，还涉及到一些服务端知识，可以快速了解全栈开发流程；同时也是个人的学习笔记，希望对大家有用。
:::
